### 按位非（~）

- 它将操作数转化为 32 位的有符号整型，运算公式 <font color=red>- ( x + 1)</font>

    ```js
    ~null;      // => -1
    ~undefined; // => -1
    ~0;         // => -1
    ~{};        // => -1
    ~[];        // => -1
    ~(1/0);     // => -1
    ~false;     // => -1
    ~true;      // => -1
    ~1.2543;    // => -2
    ~4.9;       // => -5
    ~(-2.999);  // => 1
    ```
- 由于数字 ~(-1) 和 ~4294967295（2^32 - 1）均使用 32 位表示形式，它们的运算结果均为 0。
   ```js
   ~(4294967295) // 0
   ``` 
### 双重按位非（~~）

- 双重按位非实际就是对按位非再做一次按位非，及运算公式 <font color=red>-(-( x + 1 )) + 1</font>

    ```js
    ~~null;      // => 0
    ~~undefined; // => 0
    ~~0;         // => 0
    ~~{};        // => 0
    ~~[];        // => 0
    ~~(1/0);     // => 0
    ~~false;     // => 0
    ~~true;      // => 1
    ~~1.2543;    // => 1
    ~~4.9;       // => 4
    ~~(-2.999);  // => -2
    ```
- 由于数字 ~(-1) 和 ~4294967295（2^32 - 1）均使用 32 位表示形式，它们的运算结果均为 0。
   ```js
   ~~4294967295 // -1
   ``` 

### 双重按位非（~~）与 Math.floor() 相似

- Math.floor()总是返回小于等于一个给定数字的最大整数
 
    ```js
    Math.floor(null)        // => 0 
    Math.floor(undefined)   // => NaN
    Math.floor(0)           // => 0
    Math.floor({})          // => NaN
    Math.floor([])          // => 0
    Math.floor((1/0))       // => Infinity
    Math.floor(false)       // => 0
    Math.floor(true)        // => 1
    Math.floor(1.2543)      // => 1
    Math.floor(4.9)         // => 4
    Math.floor(-2.99)       // => -3
    Math.floor(-2.29)       // => -3
    
    ```
